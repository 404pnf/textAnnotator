textAnnotator

It is a small and simple JS tool for making powerful underline, highlight and strike text annotations

What It Can Do

  • Underline, hightlight or strike text.
  • Annotated selected text (if rangy included).
  • Any color you want.
  • Extensible templates
  • IE9+, Chrome, FireFox, Andorid 4+, iOS are suppoted.
  • IE6+ support can be implemented (using Raphael.JS)

Take a try:

Select text and press underline, highlight or strike

How To Use:

  • Download
  • For basic functions add links to the following css & js files:
    • css/annotator.css
    • build/annotator.min.js
  • For extended features and possibilities:
    • Rangy framework
    • js/AnnotatorExtra.js - for some extra templates

Simple annotation:

var u = new tvs.Annotator();
u.underline(element1, 'solid', 'red');
u.highlight(element2, 'brush', 'green');
u.strike(element3, 'dashed', 'blue');
